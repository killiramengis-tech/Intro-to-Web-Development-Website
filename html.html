HTML Cheat Sheet

Common HTML elements I learned and use when building websites.

<html>

Root element of an HTML document.

<html> ... </html>

<head>

Contains metadata like title and links.

<head> ... </head>

<body>

Holds all visible content.

<body> ... </body>

<h1> – <h6>

Headings from largest to smallest.

<h1>Title</h1>

<p>

Paragraph text.

<p>Hello world</p>

<a>

Creates a hyperlink.

<a href="page.html">Link</a>

<img>

Displays an image.

<img src="img.jpg" alt="image">

<div>

Container for layout.

<div>Content</div>

<span>

Inline container.

<span>text</span>

<ul>

Unordered list.

<ul><li>Item</li></ul>

<ol>

Ordered list.

<ol><li>Item</li></ol>

<li>

List item.

<li>Item</li>

<button>

Clickable button.

<button>Click</button>

<input>

User input field.

<input type="text">

<form>

Collects user input.

<form>...</form>
