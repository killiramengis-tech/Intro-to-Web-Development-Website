CSS Cheat Sheet

Common CSS properties used to style websites.

color

Changes text color.

color: red;

background-color

Sets background color.

background-color: blue;

font-size

Changes text size.

font-size: 16px;

font-family

Changes the font.

font-family: Arial;

margin

Space outside an element.

margin: 10px;

padding

Space inside an element.

padding: 15px;

border

Adds a border.

border: 1px solid black;

width

Sets element width.

width: 200px;

height

Sets element height.

height: 100px;

display

Controls how an element displays.

display: flex;

flex-direction

Direction of flex items.

flex-direction: row;

justify-content

Aligns items horizontally.

justify-content: center;

align-items

Aligns items vertically.

align-items: center;

box-shadow

Adds shadow to elements.

box-shadow: 0 4px 10px gray;

border-radius

Rounds corners.

border-radius: 8px;
